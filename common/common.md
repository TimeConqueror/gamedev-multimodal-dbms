# Построение: 
Коллекция players, пример документа:
```json
{
  "inventory": [
    {
      "item": "items/stone",
      "count": 5,
      "data": {}
    },
    {
      "item": "items/wood",
      "count": 6,
      "data": {}
    }
  ],
  "params": {
    "pos": [
      100,
      200,
      300
    ],
    "level": 54,
    "experience": 58000
  },
  "online": true
}
```

Коллекция предметов, пример документа:
```json
{
  "name": "Iron Sword",
  "desc": "The Sword to make enemies cry!",
  "meta": {
    "durability": 260
  }
}
```
